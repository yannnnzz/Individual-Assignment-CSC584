# Brief Project Description

This project is a Personal Profile Management System developed using HTML, JSP, Java Servlet, JavaBean, JDBC, and Java DB (Apache Derby) following the MVC architecture. The system allows users to store and manage profile information in a database.


# Implemented Features

# Main Features
- Create Profile
- Display Profile Information
- Store Profile Data into Database
- View All Profiles

# Additional Feature
- Delete Profile
